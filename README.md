# 🧠 Next Word Prediction using LSTM (Streamlit App)

## 📌 Overview
This project is a Deep Learning-based web application that predicts the next word in a sentence using an LSTM (Long Short-Term Memory) model. The model is trained on text data and deployed using Streamlit.

---

## 🚀 Features
- Predicts next word based on user input
- LSTM-based sequence prediction
- Interactive Streamlit UI

---

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
